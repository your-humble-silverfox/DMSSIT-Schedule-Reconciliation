# Сверка Расписания

Программа выполняет сверку расписания поточных лекций и дисциплин с нагрузкой преподавателей на год, т.е. закрепление лекционных потоков и практических занятий за конкретными преподавателями. Программа сопоставляет файл расписания и файл нагрузки, выводя перечень несовпадений с указанием необходимых замен, а также перечень занятий, где не указаны преподаватели вовсе.

## Использование приложения

Приложение представляет собой python-скрипт, запускаемый из командной строки следующим образом:

```bash
python schedule_reconciliation.py [путь к таблице с расписанием] [путь к таблице с нагрузкой]
```

**Важно:** Таблица с расписанием должна быть в формате `.xlsx`
